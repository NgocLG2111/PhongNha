# Menu image files

Image structure:

```text
assets/images/menu/{menu-type}_{category}_{dish}.webp
assets/images/menu/{menu-type}_category_{category}.webp
```

Rules:

- Put every image directly inside `assets/images/menu/`.
- Do not create category folders.
- Dish image format: `{menu-type}_{category}_{dish}.webp`.
- Category image format: `{menu-type}_category_{category}.webp`.

Examples:

```text
assets/images/menu/main-menu_velke-polievky_ramen.webp
assets/images/menu/lunch-menu_piatok-polievky_hanoj-vyvar.webp
assets/images/menu/lunch-menu_category_pondelok-teple-jedla.webp
```
